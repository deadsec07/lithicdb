Disk-First

Vectors stay in generation files on disk while the in-memory footprint stays centered on routing, filters, and handles.

ANN + Filters

Hybrid graph search with exact and range-aware metadata filtering, plus brute-force cosine benchmarking from the same storage state.

Rust Core

Single-repo Rust implementation with no dependency on external vector database engines or FAISS-class ANN cores.

Why It Exists

Pinecone, Qdrant, Weaviate, and Milvus set the bar, but there is still room for a sharply positioned single-node product built around lower memory pressure and straightforward operational control.

  • Compressed payload vectors live on disk.
  • Graph routing and metadata indexes stay in memory.
  • WAL, diagnostics, backup/restore, and generation publishing are built into the repo.

What You Get

  • Create collections with fixed dimensions.
  • Insert, delete, fetch, and search vectors through REST.
  • Filter with exact terms and numeric ranges.
  • Run CI on pushes to main and publish this page through GitHub Pages.
